ContactCards

ContactCards is a simple address book written in C.

vCards are safed in a local database and fetched from a remote server using the cardDav protocoll.

Bugtracker

Requirements

The following packages are required:

  • GTK+
  • SQLite
  • neon

Build

$ ./autogen.sh
$ ./configure
$ make

Features

  • CardDAV: vCard Extensions to Web Distributed Authoring and Versioning (WebDAV) RFC 6352
  • The OAuth 2.0 Authorization Framework RFC 6749
  • HTTP Authentication: Basic and Digest Access Authentication RFC 2617
  • Using POST to Add Members to Web Distributed Authoring and Versioning (WebDAV) Collections RFC 5995
  • vCard Format Specification (Version 4) RFC 6350

Try it!

Install the dependencies.

On Fedora run:

$ yum install autoconf gettext-devel automake intltool gcc git gtk3-devel neon-devel sqlite-devel

On Ubuntu run:

$ apt-get install autoconf autopoint intltool gcc git libgtk-3-dev libsqlite3-dev libneon27-dev

Now get it, build it and run it:

$ git clone https://github.com/florianl/ContactCards.git
$ cd ContactCards
$ ./autogen.sh
$ ./configure
$ make
$ ./src/contactcards

Alternatively ContactCards may be also installed from copr using copr.fedoraproject.org/coprs/flo/contactcards.
